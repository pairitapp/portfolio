Asset Allocation

I created the Asset Allocation project was created to learn more about the applications of computer science and mathematics and the world of finance using historical stock data and the Sharpe ratio and the Monte Carlo method.

The scatterplot shows the annualized return and the standard deviation of each portfolio generated through the Monte Carlo method. This plot demonstrates the concept of the efficient frontier in portfolio optimization.

The second plot is an example of the performance of one of the portfolios that was generated.

When ran, the program prints out a table of the portfolios sorted by their Sharpe ratio. We can see in that table the weight the various stocks have to obtain this ratio.

Technologies used
yfinance, matplotlib, numpy, pandas
